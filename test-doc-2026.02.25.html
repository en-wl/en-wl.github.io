This form can be used to test that the ESDB based speller dictionaries are installed correctly if you are using a browser. If you are using LibreOffice you can use this doc: test-doc-2026.02.25.odt. If you are using some other application than copy and paste the text inside the text area below.

For the default (smaller) dictionary none of all words under added should not be marked, while all the words under removed should be. For the larger dictionary alternative spellings such as “Kiev” should not be marked, as that size let’s in some common spelling variants.
